Worksheet Studio

paste problems → typeset PDF
Input format 
Line 1        → title
Line 2        → subtitle (optional)
Problem 1.    → starts a problem
              (also: 1.  1)  Q1.  #1)
*Problem 5.   → starred challenge problem
(a) ... (b)   → lettered parts
Answer Key    → everything after goes to
                a boxed key on a new page
Math goes in $...$ as usual.

Paste a title and problems on the left, then select Generate worksheet to preview the pages here.
